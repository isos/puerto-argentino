// 2007-12-04

This is a major change release. The number of files affected was significant (over 50%). The fundamental structure was re-designed for a more modular/standardized approach.

Improved hooks for customization. Allows for personalization without affecting the trunk for easier updates.

Improved paging capability including search parameter retention for return to prior state.

Added ability to view reports/forms in popup window, download in PDF, or email directly from PhreeBooks.

Chagned license to GPL Rev. 3

Fixed more bugs as they were reported and discovered during testing.
