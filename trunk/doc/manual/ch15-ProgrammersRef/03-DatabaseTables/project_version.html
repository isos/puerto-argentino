Table project_version

Table project_version hold the database release information. This data is compared every time a user logs into PhreeBooks the the project release. If the database release is lower, a script is executed to update the database to the same level as the project.

Table Type: InnoDB

Field Type Default Description
project_version_id int (11) NOT NULL PRIMARY KEY - Auto increment field to hold unique ID
project_version_key varchar(40)   Module/Project name
project_version_major varchar(20)   Major release number
project_version_minor varchar(20)   Minor release number
project_version_comment varchar(250)   Release Comments
project_version_date_applied datetime   Timestamp of the record insert/change
