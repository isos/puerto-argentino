Welcome To PhreeBooks™ - Business Mangement Software

Designed for Small Business


PhreeBooks ™ is a free, user-friendly, open source acounting (ERP) system. The software's foundation is based off of the open source e-commerce package PhreeBooks and osCommerce shopping cart administrator application with a robust accounting engine and robust features to interface with outside applications. PhreeBooks was originally developed by the PhreeSoft development team to provide a web based accounting package that is easy to setup and operate for small businesses.

The goal of the PhreeBooks project is to provide a world-class accounting application equaling popular commercial packages with a development path and support network eager to meet the needs of all its users.

PhreeBooks utilizes the PhreeHelp popup healp system to provide detailed help for every aspect of the program, from installation, setup, detailed program information to a easy to navigate users guide.

But most importantly, it's FREE!

Designed for Online Operation

PhreeBooks is designed as an web based application. It can be set up to operate in a secure (SSL) environment hosted locally or through any web provider. Interfaces to popular shopping cart systems is included to allow easy data transfer between PhreeBooks and third party e-commerce systems.

The program provides extensive security settings to allow control of access to the users of the system. Users can be granted access (no access, read only, enter only, edit, full/delete) to almost every operation available.

PhreeBooks has an extensive report builder capability to not only modify any of the several standard reports but build you own to display only the information the you want. Additionally, the report builder creates forms for invoicing, purchases, and banking operations.

Next, a payment type is chosen from one of the popular payment modules. (PayPal™ and AuthorizeNet™ are just 2 of the included modules).

PhreeBooks™ addresses many of the goals we set:

  • Make the program easy to set up
  • Make it easy to customize
  • Integrate with other applications (e-commerce and CRM)
  • Make navigation simple
  • Manage customers and vendors, purchase and sales orders including many shipping options, inventory management, import/export of information, banking and extensive reporting
  • Expandable inventory with customizable atributes and a multilevel pricing manager
  • Password protected with extensive security to control user access to certain modules, can operate in an SSL environment
  • Support multiple languages, currencies and tax structures

For Web Designers


PhreeBooks™ is based off of the admin menu structure of the highly successful osCommerce/ZenCart e-commerce applications. The style sheets are compatible between PhreeBooks and ZenCart. Because of this, many of the PhreeBooks menus have the same look and feel as the ZenCart admin functions.

PhreeBooks™ contains the PhreeHelp popup menu help system with searchable capoabilities and an extensive write-up of the program structure and organization. Additonally, the database tables are detailed in the documentation to help the technbical user understand each table element and how it is used within the application.

For Developers and Hosting Providers

Developers are able to deploy cutting-edge solutions based on PhreeBooks™'s modern standards and technologies. Combining PHP server-side technology with the ability to choose from multiple databases on the back end results in flexible software that you can customize and modify to suit your needs.

Minimum/suggested server requirements:

  • PHP 4.1.2(patched)+ or higher (PHP 4.3.3+ for optimal performance)
  • Apache 1.3.x or higher
  • MySQL 3.2x.x or higher
  • PhreeBooks™ requires "register globals" off.
