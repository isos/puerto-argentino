Bienvenido a PhreeBooks™ - Programa de Computación para la Administración de Negocios

Diseñando para la Pequeña Empresa


PhreeBooks ™ es un programa de contabilidad de fuente libre del tipo Planificación de Recursos Empresariales (ERP) con excelente usabilidad. La base del programa es el comercio en línea de fuente libre y el administrador de la aplicación osCommerce con un motor de contabilidad y características robustas para interfazar con otras aplicaciones. PhreeBooks fue desarrollado originalmente por el equipo de desarrollo de PhreeSoft para proveer una aplicación de contabilidad basado en red que fuera fácil de instalar y operar para las empresas pequeñas.

La meta del proyecto PhreeBooks es proveer una aplicación de contabilidad de clase mundial comparable a los paquetes comerciales populares con un equipo de desarrollo y una base de soporte orientado a satisfacer las necesidades de los usuarios.

PhreeBooks utiliza PhreeAyuda, un sistema de pantallas plegables sobre aspectos del programa, desde la instalación, configuración, y detalles sobre el uso del programa hasta una guía de usuario fácil de navegar.

Pero sobre todo, es GRATIS!

Diseñado Para Operar en Línea

PhreeBooks está diseñado como una aplicación basada en red. Puede operar en un ambiente seguro (SSL), hospedado localmente o en cualquier servidor en internet. Tiene interface con sistemas populares de carro de compras para permitir la fácil transferencia de información entre PhreeBooks y los sistemas de comercio en línea.

El programa trae extensas opciones de seguridad para controlar el acceso al sistema por parte de los usuarios. A los usuarios se les puede dar permisos de acceso (ningún acceso, solo lectura, ingreso solamente, para modificar, completa/borrar) a casi todas las funciones disponibles.

PhreeBooks viene con un poderoso Constructor de Reportes no solo para modificar los reportes modelos que trae sino también para construir sus propios reportes y mostrar solamente la información que necesite. Además, el Constructor de Reportes crea formularios para las facturas, compras y operaciones con los bancos.

Se escoje una forma de pago de entre algunos de los módulos de pago conocidos. (PayPal™ y AuthorizeNet™ son solo algunos de los módulos incluidos).

PhreeBooks™ Responde a Muchas de las Metas Fijadas:

  • Hacer el programa fácil de montar
  • Hacerlo fácil de personalizar
  • Integrarlo con otras aplicaciones (comercio en línea y CRM)
  • Hacer la navegación simple
  • Administrar Listas de Clientes y Proveedores, Órdenes de Venta y Compra incluyendo opciones de envío, administración de inventario, importar/exportar información, banca y toda clase de reportes
  • Inventario expandible con atributos personalizados y un administrador de precios multi-nivel
  • Protegido por clave con un extenso sistema de seguridad para controlar el acceso de los usuarios por módulo y puede operar en un ambiente SSL
  • Soporta múltiples idiomas, monedas y estructuras de impuestos

Para Diseñadores de Sito


PhreeBooks™ esta basado en una estructura de menúes de los populares programas de comercio en internet osCommerce/ZenCart. Las hojas de estilo CSS son compatibles entre PhreeBooks y ZenCart. Por esta razón, muchos de los menúes de PhreeBooks tienen la misma apariencia y uso que en las funciones del administrador de ZenCart.

PhreeBooks™ contiene el sistema de ayuda PhreeAyuda con capacidad de búsqueda de la estructura y organización del programa. Además, las tablas de la base de datos están detalladas en la documentación para facilitarle al usuario técnico entender los elementos de cada tabla y cómo estan siendo usado en la aplicación.

Para Programadores y Administradores de Servidores

Los programadores pueden hacer uso de soluciones con estándares modernos y tecnología de punta como son usados por Phreebooks™. Combinado con procedimientos de tecnología PHP en el servidor, con la abilidad de escoger entre múltiples bases de datos, provee un programa flexible que se puede personalizar y modificar a sus necesidades.

Requisitos Mínimos/Sugerencias para el Servidor:

  • PHP 4.1.2(con parches)+ o mayor (PHP 4.3.3+ para desempeño óptimo)
  • Apache 1.3.x ó mayor
  • MySQL 3.2x.x ó mayor
  • PhreeBooks™ requiere "register globals" "off".
